# ReduxDoubleTrouble
React, Redux, Gulp

### Checkout

:computer: Checkout this repo, install dependencies, then start the http-server process with the following:

```
	> git clone git@github.com:AceDZN/ReduxDoubleTrouble.git
	> cd ReduxDoubleTrouble
	> npm install
	> gulp serve
```
